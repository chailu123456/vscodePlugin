 quick-push-code

## 插件介绍

是一款一键推动代码插件，使用shelljs执行一系列git操作，将代码推送到远程仓库，提升效率

点击鼠标右键, 一键推送代码点击后会弹出输入框,输入commit信息(可不填), 默认 commit = "perf: update"
```
commit 规范

feat 增加新功能
fix 修复问题/BUG
style 代码风格相关无影响运行结果的
perf 优化/性能提升
refactor 重构
revert 撤销修改
test 测试相关
docs 文档/注释
chore 依赖更新/脚手架配置修改等
workflow 工作流改进
ci 持续集成
types 类型定义文件更改
wip 开发中

commit提交格式为：
git commit -m 'xxx: 你的修改信息'


打包为npm run build

提示：如果是首次推送代码到远程，要先关联远程,则要手动进行关联
```
