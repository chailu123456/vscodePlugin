 quick-push-code

## 插件介绍

是一款一键推动代码插件，使用shelljs执行一系列git操作，将代码推送到远程仓库，提升效率

点击鼠标右键, 一键推送代码点击后会弹出输入框,输入commit信息(可不填)

打包为npm run build

提示：如果是首次推送代码到远程，要先关联远程,则要手动进行关联
